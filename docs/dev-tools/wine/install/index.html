    How to install the xPack WineHQ binaries

    The recommended method is via xpm.

    Overview

    The xPack WineHQ can be installed automatically, via xpm (the recommended method), or manually, by downloading and unpacking one of the portable archives.

    • Windows
    • macOS
    • GNU/Linux

    Binaries are available only for GNU/Linux.

    Binaries are available only for GNU/Linux.

    Easy install

    The easiest way to install WineHQ is by using the binary xPack, available as @xpack-dev-tools/wine from the npmjs.com registry.

    Prerequisites

    The only requirement is a recent xpm, which is a portable Node.js command line application. To install it, follow the instructions from the xpm install page.

    Install

    With xpm available, installing the latest version of the package is quite easy:

    xpm install --global @xpack-dev-tools/wine@latest --verbose

    This command will always install the latest available version, in the global xPacks store, which is a platform dependent folder (check the output of the xpm command for the actual folder used on your platform).

    Tip: The install location can be configured using the XPACKS_STORE_FOLDER environment variable; for more details please check the xpm folders page.
    Tip: The archive content is unpacked into a folder named .content. On some platforms this might be hidden for normal browsing, and require separate options (like ls -A) or, in file browsers, to enable settings like Show Hidden Files.

    Uninstall

    To remove the installed xPack, the command is similar:

    xpm uninstall --global @xpack-dev-tools/wine --verbose

    Test

    To check if the xpm installed WineHQ starts, use something like:

    $ ~/.local/xPacks/@xpack-dev-tools/wine/7.22.0-1.1/.content/bin/wine64 --version
wine-7.22

    Manual install

    For all platforms, the xPack WineHQ binaries are released as portable archives that can be installed in any location.

    The archives can be downloaded from the GitHub releases page.

    Download

    The GNU/Linux versions of xPack WineHQ are packed as .tar.gz archives. Download the latest version named like:

    • xpack-wine-7.22.0-1-linux-x64.tar.gz

    As the name implies, these are GNU/Linux tar.gz archives; they were build on Ubuntu, but can be executed on most recent GNU/Linux distributions.

    Unpack

    To manually install the xPack WineHQ, unpack the archive and move it to ~/.local/xPacks/wine/xpack-wine-7.22.0-1:

    mkdir -p ~/.local/xPacks/wine
cd ~/.local/xPacks/wine

tar xvf ~/Downloads/xpack-wine-7.22.0-1-linux-x64.tar.gz
chmod -R -w xpack-wine-7.22.0-1
    Note: For manual installs, the recommended install location is slightly different from the xpm install folders, which use the scope (like @xpack-dev-tools) to group different tools, and .content to store the unpacked archive.
